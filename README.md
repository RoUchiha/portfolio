# Roshaan Singh — Portfolio

Personal portfolio site for Roshaan Singh, AI/ML and full-stack software engineer
focused on LLM evaluation, AI reliability, observability, and production GenAI systems.

**Live:** https://portfolio-rouchihas-projects.vercel.app

## Stack

Static site — plain HTML, CSS, and vanilla JS. No build step.

- `index.html` — page markup and content
- `styles.css` — styling
- `script.js` — scroll-reveal + animated signal-network canvas background
- `assets/` — hero image
- `Roshaan_Singh_Resume_2026.pdf` — downloadable resume

## Develop locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy

Hosted on Vercel. Pushing to `main` triggers a production deploy.
